# 👨‍💻 Portfolio - Marcel Welk

Mein persönliches Portfolio als IT-Techniker & Developer.

**Live:** [portfolio-fawn-seven-37.vercel.app](https://portfolio-fawn-seven-37.vercel.app)

---

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **UI:** shadcn/ui
- **Icons:** Lucide
- **Hosting:** Vercel

---

## 🚀 Lokal starten

```bash
# Dependencies installieren
npm install

# Dev Server starten
npm run dev
```

Öffne `http://localhost:3000`

---

## 📁 Struktur

```
portfolio/
├── app/              # Next.js App Router
├── components/       # React Komponenten
│   ├── hero.tsx      # Hero Section
│   ├── projects.tsx  # Projekte
│   ├── skills.tsx    # Skills & Zertifikate
│   ├── contact.tsx   # Kontakt
│   └── ui/           # shadcn/ui Komponenten
├── public/           # Statische Assets
└── styles/           # CSS
```
